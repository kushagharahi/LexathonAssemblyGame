Lexathon
========

Final Game Written And Designed By:
	

(A Simple and Addictive Word Game!)

To RUN:
1. Make sure all the included files are in the SAME folder, and not scattered around, (In other words, just download the .zip and extract)

2. Open MARS4_4 from inside the Lexathon folder

3. Open runprog.asm

4. In the settings menu in MARS, make sure "Initialize Program Counter to global 'main' if defined" is checked

5. ASSEMBLE the program (little button on top with the wrench)

6. Press Play, and enjoy!!



Note: 	For best results, close and re-open MARS after completing the game.

Limitation: Not currently supported on Linux
